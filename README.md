# DD-WRT Automatic Updater

### Requirements

- A Linux box with network access to the DD-WRT device(s).

### Installation

- Clone the repository. Suggested location: `$HOME/opt/`.
- Follow configuration instructions in `ddwrt-update.TEMPLATE`.
